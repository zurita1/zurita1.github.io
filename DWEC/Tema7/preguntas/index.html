AJAX con jQuery

  • El método ajaxSetup() establece los valores por defecto en las peticiones Ajax. Indica por qué su uso no está recomendado.

  • Indica los 6 manejadores de eventos globales de AJAX.

    • .ajaxComplete(),Registra un controlador para que se llame cuando se completen las solicitudes de Ajax.

    .ajaxError(),Registra un controlador para que se llame cuando Ajax solicite un error. Este es un evento de Ajax.

    .ajaxSend(),Añade una función que se ejecutará antes de enviar una solicitud Ajax. Este es un evento de Ajax.

    .ajaxStart(),Registra un controlador para que se llame cuando comience la primera solicitud de Ajax. Este es un evento de Ajax.

    .ajaxStop(),Registra un controlador para que se llame cuando todas las solicitudes de Ajax se hayan completado. Este es un evento de Ajax.

    .ajaxSuccess(),Añade una función para que se ejecute cada vez que una solicitud Ajax se complete con éxito. Este es un evento de Ajax.

  • Indica al objeto al que han de estar asociados los métodos anteriores.

    • Al objeto DOM que se especifique o AJAX.

  • Los métodos AJAX abreviados son 5. Indica cuáles son y utilidad de cada uno.

    • .get(),Carga los dato del servidor usando una solicitud HTTP GET.

    $.get( "test.php", function( data ) {
    $( "body" )
    .append( "Name: " + data.name ) // John
    .append( "Time: " + data.time ); // 2pm
    }, "json" );

    .getJSON(),Carga datos JSON codificados desde el servidor mediante una solicitud HTTP GET.

    $.getJSON( "test.js", { name: "John", time: "2pm" } )
    .done(function( json ) {
    console.log( "JSON Data: " + json.users[ 3 ].name );
    })
    .fail(function( jqxhr, textStatus, error ) {
    var err = textStatus + ", " + error;
    console.log( "Request Failed: " + err );
    });

    .getScript(),Carga un archivo JavaScript del servidor usando una solicitud HTTP GET, luego se ejecuta.

    var url = "https://code.jquery.com/color/jquery.color.js";
    $.getScript( url, function() {
    $( "#go" ).click(function() {
    $( ".block" )
    .animate({
    backgroundColor: "rgb(255, 180, 180)"
    }, 1000 )
    .delay( 500 )
    .animate({
    backgroundColor: "olive"
    }, 1000 )
    .delay( 500 )
    .animate({
    backgroundColor: "#00f"
    }, 1000 );
    });
    });

    .post(),Carga los datos del servidor usando una solicitud HTTP POST.

    $.post( "test.php", { name: "John", time: "2pm" })
    .done(function( data ) {
    alert( "Data Loaded: " + data );
    });

    .load(),Carga los datos del servidor y coloca el HTML devuelto en el elemento coincidente.
